Inline vs Block-Level elements

Inline

List of a couple of inline elements

  • <q>
  • <strong>
  • <b>
  • <i>
  • <em>

Inline elements will keep the content inline and not create a newline

Block-Level

list of a couple block-level elements

  • <blockquote>
  • <p>
  • <ul>
  • <ol>
  • <dl>

Block level elements will create a new line and occupy its own block for it's content.

You can use CSS to switch any elements layout from 'inline' to 'block' and visa versa.
